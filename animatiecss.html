3D CSS Animatie

Geanimeerde 3D met enkel code

Opdracht:

Maak een 3D Animatie met alleen CSS en wat basis HTML.

Focus:

CSS

Proces:

Met behulp van een youtube tutorial meecoderen en door kleine aanpassingen te maken meer te snappen van wat er wordt uitgelegd. Op codepen een nieuw project aangemaakt en tegelijk met de video aan het project werken.

Link naar Project

https://codepen.io/maartenvnstra/pen/XWeoGMR

Reflectie

Ik vind spelen met de limieten van CSS enorm interessant. 3D in de browser met CSS lijkt me leuk om vaker toe te passen. Vergeleken met het JS project (Boter, Kaas en Eieren) heb ik dit meer onder controle. Meecoderen met tutorials is voor mij een goede manier om snel nieuwe principes te begrijpen.
