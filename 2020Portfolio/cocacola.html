Coca Cola Website

Bestaande site, schonere code

Opdracht:

Maak een bestaande site na met zo min mogelijk divs en classes/id's. Maak deze site toegankelijk voor mensen met een beperking.

Focus:

HTML, CSS, JS, UX

Proces:

Met behulp van de inspector de assets van de bestaande site getrokken en vervolgens een compleet nieuwe site gemaakt met HTML, CSS en Javascript. Elementen toegevoegd waarmee de gebruiker door de hele site kan navigeren met de Tab knop en een theme switch toegevoegd waarmee een dark-mode aan en uit gezet kan worden.

Link naar Project

https://github.com/Dobbyd/BlokWeb2020

Reflectie

Deze site maakt optimaal gebruik van css selectoren en enkele obscure trucjes en is hierdoor semantisch bijna perfect. Ook is de site volledig responsive voor ieder device. Aangezien ik niet enorm ervaren ben met coderen ben ik erg tevreden over het eindresultaat. Ik denk hiermee een goede basis te hebben gelegd voor projecten die werken met enkel HTML, CSS en JS.
